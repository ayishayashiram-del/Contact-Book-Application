# 📒 Contact Book Application

## 📌 Project Overview

This is a **Contact Book Application** developed using **Python** as part of my ** Python Programming**.

The application helps users store and manage contact information efficiently through a simple command-line interface. Contact details are automatically saved in a JSON file, so they remain available even after restarting the program.


## ✨ Features

- ➕ Add New Contact
- 📋 View All Contacts
- 🔍 Search Contact
- ✏️ Update Contact
- ❌ Delete Contact
- 📊 Contact Summary
- 💾 Automatic Data Saving (JSON)
- 📂 Automatic Data Loading
- ⚠️ Error Handling


## 🛠️ Technologies Used

- Python 3
- File Handling
- Dictionaries
- Loops
- Conditional Statements
- Functions
- Exception Handling

## ▶️ How to Run

1. Install Python 3.
2. Download the project.
3. Open Terminal or Command Prompt.
4. Run the following command:

```bash
python contact_book.py
```
